# AQA Python Portfolio – Yuliia Valkovets

This repository contains practical tasks and training projects from the QA Automation (Python) course, demonstrating my skills in Python, test automation, API testing, and core QA tools

---

## Key skills: 
Python automation, Selenium (web UI), Pytest (unit & API tests, fixtures, parametrization), Page Object Model, XPath/CSS selectors, HTTP requests (GET/POST/PUT/DELETE), logging.

---

## Recommended Projects

Web UI Automation – Selenium tests for web interfaces (lesson_27) demonstrating Page Object Model, locators, and user scenarios.

API Testing – Pytest API tests (lesson_10 / lesson_15) with fixtures, parameterization, and response validation.

Data Handling – Scripts for JSON/CSV processing and test data preparation (lesson_13).

Test Frameworks & Logging – Using unittest/pytest with test logging (lesson_10, lesson_11).

Python Fundamentals for QA – Core Python concepts: functions, OOP, decorators, generators, Git, and virtual environments (lesson_01–lesson_09).

## How to run the code

git clone https://github.com/YuliaValkovets/aqa_python_valkovets.git

cd aqa_python_valkovets

python -m venv venv && source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt

pytest lesson_27/ lesson_10/ lesson_09/  # Run Web UI, API, and Unit tests
